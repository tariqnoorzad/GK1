# FindTickets 

En Expo/React Native app udviklet som en del af Godkendelsesopgave 1.  

## Installation
1. Klon projektet:
   git clone https://github.com/tariqnoorzad/GK1.git
   cd GK1/FindTickets

Installer dependencies:
npm install
npx expo start
